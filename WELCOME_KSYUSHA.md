## Первый день (60-90 минут)

### 1. Установи инструменты (если нет)

```bash
# macOS
brew install git gh node@22
gh auth login   # через GitHub, выбери HTTPS
```

### 2. Склонируй репо

```bash
mkdir -p ~/Sites && cd ~/Sites
git clone https://github.com/antonoreshkin/anton-site.git
cd anton-site
npm install
git config core.hooksPath .githooks   # включает защиту main
```

### 3. Запусти локально

```bash
npx next dev --webpack
# открой http://localhost:3000 в Chrome
# протестируй мобильный режим (DevTools → iPhone 14)
```

### 4. Прочитай в таком порядке

1. **`README.md`** — что за проект, в двух абзацах
2. **`CLAUDE.md`** — стиль, табу, контекст Антона (важно!)
3. **`COLLABORATION.md`** — как работаем втроём (ты, Антон, Claude)
4. **`HANDOFF.md`** — техническое состояние v2.3
5. **`docs/handover/`** — глубокие документы:
   - `01_TECH_AUDIT.md` — тех-стек и план развития
   - `02_DEVOPS_SERVICES.md` — все сервисы и доступы
   - `03_CONTENT_GUIDE.md` — как наполнять контент + будущий Sanity
   - `04_PMO_REGLAMENT.md` — регламент работы

### 5. Сделай первую правку (тренировочную)

```bash
git checkout -b content/first-commit-ksyusha
# открой src/lib/content.ts, поправь любую опечатку
git add -A
git commit -m "первая правка для проверки workflow"
git push -u origin content/first-commit-ksyusha
gh pr create --fill
```

Дождись Vercel preview-комментария в PR (~2 минуты), посмотри что превью работает, нажми **Merge** (squash). Готово — твоя правка на проде через ещё ~2 минуты.

## Эскалация к Антону — когда писать в Telegram

- Меняется **позиционирование** Антона (как он представляется, миссия, стратегия)
- Новая **категория** проектов (помимо commercial / social)
- Правка может затронуть **табу**: Алексей Третьяков, ИРИ, Атланты Сити, Signal TV, AGORA, доли, суммы
- Запросы от **третьих лиц** (партнёры, журналисты) — Антон решает что отвечать
- Технические **затраты выше 1000 ₽/мес** (например апгрейд Vercel Pro, покупка домена)
- Что-то на проде сломалось и ты не уверена как чинить → пиши, починим вместе

## Стиль — самое короткое

| Запомни | Пример |
|---|---|
| АССИСТ+ | заглавными, не «Ассист+» |
| Oreshkin Media Lab | на английском |
| X Forum | не «Иксфорум» |
| Иван Гуз | через «з», не Гус |
| Голос | сдержанно, на «вы», искренне |
| Не использовать | длинные тире, «данный», «является», «уникальная возможность», «успешный успех» |
| Не упоминать | Третьяков, ИРИ, Атланты, Signal TV, конкретные доли и суммы |

Полная версия — в `CLAUDE.md` и `docs/handover/03_CONTENT_GUIDE.md`.
