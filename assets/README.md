# Ассеты

Локальное зеркало материалов из Google Drive [12_Сайт](https://drive.google.com/drive/folders/1TwefqekaGZVK92avHFDIxNpuJtStE1Je).

## Структура

- `photos/` — фотографии Антона (портреты, рабочие моменты, мероприятия, со съёмок фильма про МФТИ, «100 капибар»).
- `logos/` — SVG-логотипы МФТИ, Сколково, Т-Банка, Физтех Союза, Орешка Медиалаб, АССИСТ+, Autonomy AI.
- `videos/` — тизер фильма про МФТИ, любые рилсы и кружки.
- `screenshots/` — скриншоты dev-сборок и эталоны (2026-05-12_smoke-1440.png — первый smoke).

## Правила

- В `public/` Next.js кладём только финальные ресайз-варианты (через `next/image`).
- Оригиналы (RAW, большие JPG, исходники видео) — здесь и в Google Drive.
- Имена файлов кириллицей **не** использовать — латиница + дефис.
- Перед коммитом — оптимизация (squoosh для JPG/PNG, FFMPEG для видео).
