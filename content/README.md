# Контент

Здесь живут текстовые драфты до того как уйдут в CMS (Sanity).

## Структура

- `copy-drafts/` — драфты больших блоков (Hero, Кто я, Медиа, Контакт).
- `projects/` — текст по каждому проекту, файл на проект:
  - `orechka-medialab.md`
  - `it-studio.md`
  - `autonomy-ai.md`
  - `assist-plus.md`
  - `mentoring.md`
  - `coffee-shops.md`
  - `donation-platform.md`
  - `school-return.md`
  - `volunteering.md`
- `stories/` — длинные сторителлинг-фрагменты (биография, ЕГЭ, грант, переезд, миссия).

## Процесс

1. Драфт пишется здесь Claude (после получения ответов от Антона).
2. Антон комментирует / правит.
3. Финальная версия копируется в Sanity (Этап 7).
4. На сайте контент читается из Sanity через next-sanity.
