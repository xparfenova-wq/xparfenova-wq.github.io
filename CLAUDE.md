# anton-site — Визитная карточка Антона Орешкина

**Стек:**
- Next.js 16.2.6 (App Router, Turbopack) — ⚠️ читай `node_modules/next/dist/docs/` перед использованием новых API, есть breaking changes
- React 19 · TypeScript · Tailwind CSS v4 (через `@theme` в `globals.css`)
- Framer Motion 12 (scroll-triggered fades, staggered reveals, hover transitions)
- Lucide React (иконки)
- clsx + tailwind-merge (`@/lib/cn`)

**Дизайн-система:** Humble — Precise White Lab (`.claude/skills/frontend-design/SKILL.md`). Источник правды — `/Users/anton1/Claude3/agents/reglaments/design/visual_design_system.md`.

## Чего НЕ делаем на сайте (явные табу от Антона)

- Не упоминаем доли в проектах, конкретные суммы, имена партнёров без согласования.
- AGORA — только формат «принимал участие в запуске международного делового клуба AGORA», без подробностей и долей.
- Сериал Т-Банк/ИРИ — НЕ упоминать (внутри Орешка Медиалаб).
- ИРИ как организация — НЕ упоминать.
- Атланты Сити — логотип НЕ показываем.
- Signal TV — НЕ упоминать.
- Алексей Третьяков — НЕ упоминать (просто «Студия IT-разработки»).
- Слово «курс» — разрешено (Антон снял табу 2026-05-13). Используем «курс по нейросетям» или «обучение обращению с нейросетями». Запрет остаётся на «успешный успех», «уникальная возможность», «прорыв» в инфобиз-смысле.
- Эмодзи в UI.
- «Обрывистые цифры» формата «6 учеников, 5+ проектов» — Антон считает понтовым. Описываем миссии проектов.
- Длинные тире как пауза, канцеляризмы «данный», «является».
- Никаких больших оранжевых блоков (Electric Orange только акцент), никаких generic gradients.

## Стиль контента

- Голос: **сдержанный, на «вы», искренний, душевный**. Без юмора (лёгкие моменты ок).
- Референс корпуса: Telegram-канал «Жизнь и мысли Орешкина».
- Без ИИ-маркеров (длинные тире, «данный», «является», цепочек синонимов).
- Без инфобизнес-маркеров (продажные слоганы, «уникальная возможность», слово «курс»).
- Используем: ментор, наставник, наставническая программа, практическое сопровождение, ассоциированный партнёр, сообщество.
- Не вгонять в «короткие предложения с точками» — это не главная цель. Главное — спокойствие, структура, чёткость, при этом душевность и доброта.
